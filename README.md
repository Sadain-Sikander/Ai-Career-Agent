# AI Career Analysis Agent

An intelligent, API-driven tool that analyzes unstructured resume text to extract structured career data. Built with FastAPI and powered by Google's Gemini AI.

## Features
- **Intelligent Extraction:** Uses Gemini 2.5-Flash to analyze text and extract key skills, experience levels, and career path recommendations.
- **RESTful API:** Built with FastAPI for high-performance and asynchronous request handling.
- **Structured Output:** Returns data in a clean, valid JSON format.

## Prerequisites
- Python 3.10+
- A Google Gemini API Key (get one at [Google AI Studio](https://aistudio.google.com/))
